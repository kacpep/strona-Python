Jak zainstalować Pythona.

Aby zacząć przygodę z programowaniem w Pythone trzeba go najpierw zainstalować.
Link do oficialnej strony: www.python.org następnie znajdujesz swój system operacyjny.
Kliknij Pobierz. Kolejnym krokiem jest otwarcie pobranego pliku i podążanie za wskazówkami instalatora.
Uwaga: w oknie oznaczonym jako "setup" zaznacz opcje "Add Python 3.6 to PATH" lub "Add Python to your environment variables"
Odnajdź program o nazwie "IDLE Python" i otwórz go. Kolejne instrukcje znajdziesz już pod linkiem "Ćwiczenia".
